# @uniquedivine/jiyuu

TypeScript package that mimics functionality from Rust and Golang.

## Usage

```
bun add @uniquedivine/jiyuu # or yarn add, or npm install
```

## Hacking

Install `just` to run project-specific commands.

```bash
cargo install just
```

You can view the list of available development commands with `just -ls`.

Ref: [github.com/casey/just](https://github.com/casey/just)
